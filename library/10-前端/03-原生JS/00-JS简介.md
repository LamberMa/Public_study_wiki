**2、转义**

- decodeURI( )                   URl中未转义的字符  转移url中的中文
- decodeURIComponent( )   URI组件中的未转义字符 这种连冒号和斜杠也会给你转义
- encodeURI( )                   URI中的转义字符
- encodeURIComponent( )   转义URI组件中的字符
- escape( )                         对字符串转义  下面这三个不是针对url是针对值的转义。
- unescape( )                     给转义字符串解码
- URIError                         由URl的编码和解码方法抛出
